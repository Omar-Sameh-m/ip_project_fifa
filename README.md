# Ultimate IP Project Squad (Coralando)

## Description

This is a web-based application themed around FIFA players, showcasing individual player profiles with their merchandise. The project includes a welcome page, login/signup functionality, a home page displaying player cards, a player selection interface, and detailed profile pages for each player featuring items like t-shirts, mugs, and footballs.

## Features

- **Welcome Page**: Animated introduction to the app.
- **Authentication**: Simple login and signup forms.
- **Home Page**: Displays FIFA-style cards for featured players with animations.
- **Player Selection**: Grid layout to choose from available players.
- **Player Profiles**: Detailed pages for each player, including their card and merchandise items with hover effects.
- **Responsive Design**: Mobile-friendly layout using CSS media queries.
- **Navigation**: Back buttons and links between pages.

## Installation/Setup

1. Clone the repository:
   ```
   git clone <repository-link>
   ```
2. Navigate to the project directory:
   ```
   cd ip_project_fifa
   ```
3. Open the application in your browser by opening `welcombage.html` or `home.html`.

No additional dependencies or server setup required, as this is a static HTML/CSS/JavaScript application.

## Usage

- Start by opening `welcombage.html` in your browser.
- Click "Continue without sign in" to go to the home page, or "Login" to access the login form.
- From the home page, click "Players" to select a player.
- Choose a player to view their profile with merchandise.
